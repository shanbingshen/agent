# RAG 包

`packages/rag` 是知识库处理和检索引擎。Agent 必须通过这里暴露的受控接口检索知识，不得自己实现加载、切分、Embedding 或向量库查询。

## 可以放

- 文档加载器、文本切分器、Embedding 适配、Milvus 向量库适配、检索服务和评测逻辑。
- 与知识检索有关的稳定契约。

## 不可以放

- Agent 路由逻辑。
- API 权限判断和租户授权入口。
- 设备控制、工业数据写入或 ThingsBoard 管理凭据。

## 边界

Postgres 保存知识文档、分片正文、租户和工厂权限；Milvus 保存向量、过滤字段和索引。`knowledge/` 是知识资产目录，不是向量数据库目录。
